Hi, I'm Ed Putans - a full stack software developer from London, UK.

After graduating biomedical engineering in my home country of Lithuania, I became an engineer. I discovered my love for code later - and became a different kind of engineer. I've completed the Flatiron School coding bootcamp and have been trying out different tech since.

My favourite framework is React and I get excited each time I hear about it. I enjoy building web apps using the framework and this is my go-to tool for web-development.

I feel satisfaction making web pages responsive and I love to see the page change depending on the screen size.

While I'm not coding I enjoy giving back to the community by teaching people what I know in Codebar. Living in a big city has me discovering new places every week. Longboarding is my hobby for when the weather is good and PC gaming - for when it isn't.

When I learn a new framework or feel like sharing my coding experience (or heck, when I just feel like it), I like to write down my thoughts in my blog.
